# Pizza-Sales-SQL-Window-Functions
Demonstrates advanced data analysis using SQL window functions like ROW_NUMBER(), RANK(), and SUM() OVER().

# Steps to Import dataset file in MYSQL
1.Connect to your MySQL server in MySQL Workbench
2.Go to Server → Data Import
3.Select Import from Self-Contained File
4.Choose .sql file which i have provided 
5.Click Start Import
6.Refresh the Schemas panel to see the data
